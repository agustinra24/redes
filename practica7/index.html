Actividad de Configuración de Redes con Packet Tracer

Introducción

Esta actividad tiene como objetivo proporcionar a los participantes una experiencia práctica y completa en la configuración de redes locales utilizando el programa Packet Tracer de Cisco. Los participantes simularán redes, asignarán direcciones IP (192.168.0.0) a computadoras, realizarán pruebas de conectividad con ping y registrarán visualmente el proceso mediante la grabación de videos. Más allá de reforzar conocimientos teóricos, la actividad busca desarrollar habilidades prácticas y demostrar la comprensión de conceptos de red y asignación de direcciones IP en entornos simulados.

Objetivo General

Conocer, investigar e identificar los elementos que se utilizan para configurar una red local y el acceso a recursos de un equipo en red.

Objetivos Específicos

  • Utilizar el programa Packet Tracer de Cisco para simular redes de computadora, asignando direcciones IP pertenecientes a la red 192.168.0.0 a cada computadora.
  • Emplear la ventana de comandos para utilizar la función ping y verificar la comunicación entre cada una de las computadoras en las redes simuladas.
  • Grabar un video de cada red simulada para documentar el proceso.
